This project is testing some functionality of Allmovies.com

The requirment for SUT are:

US1. Search Functionality
US2. Advanced Search by release year.

US3. Add to Favorites functionality
US4. Remove from Favorites functionality
(Log In, search Film, Add To Favorites, Check Existence in Favorite movies), (Log In, search Film, Add To Favorites, Check Existence in Favorite movies, Remove from Favorites)
US5. Account Settings - Change Name Functionality

main folder containt the POM structure pages that will be required in test cases.
The project is writen to work with chromedriver.
Firefox and Chrome drivers are in the "driver folder"
